Best sublime shortcuts

Jan 27, 2016 · 1 minute read · Comments

Following is a top collection of sublime shortcuts that are frequently used in day to day developement. Suggest something if you find anything more useful.

| Mac Command | Details | | :—————–: | : —————: | | ⌘ + P or ⌘ + T |search and open files by name |

|⌘ + R | Go to symbol. This is handy to list down method names in a file|

|⌘ + F | Find in current file |

|⌘ + ⌥ + F |Replace |

|⌘ + ⇧ + F |Find in files (This will search in the folders you specify or in all the files in general)|

|⌘ + L | Select line - Repeat to select next lines |

|⌘ + D | Select word - Repeat to select next occurrence |

| ⌃ + ⌘ + G | Select all occurrences of current selection |

|⌘ + K + ⌫ | Delete from cursor to start of line |

|⌘ + ] | Indent current line(s) |

|⌘ + [ | Un-indent current line(s) |

|⌘ + ⇧ + D | Duplicate line(s) |
